# PerspectivaHorariaMarzJul2026

Una herramienta web interactiva para construir mi horario universitario usando drag & drop.

Basado en los datos de la escuela de Ingeniería en Informática (UCAB Caracas) para el periodo marzo-julio 2026, esta página permite arrastrar secciones desde un panel lateral al horario semanal, identificar conflictos, calcular horas semanales y copiar los NRCs seleccionados.

Los horarios originales fueron recuperados de:

https://ingenieria.ucab.edu.ve/informatica/la-escuela/caracas/procesos-academicos/ingenieria-informatica/inscripciones/

## Créditos

- **Datos y diseño**: generados a partir de la perspectiva horaria publicada por la escuela.
- **Desarrollador**: Proyecto por [tu nombre/a mí] con JavaScript, HTML y CSS.
- **Inspiración**: Interfaz de horario tipo "drag & drop" popular entre estudiantes de informática.

> Si utilizas este código, atribuye adecuadamente y no dudes en adaptarlo a tu propia planificación académica.
